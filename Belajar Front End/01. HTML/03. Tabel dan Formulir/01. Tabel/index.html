Penjelasan Table dalam HTML

Table dalam HTML adalah elemen yang digunakan untuk menampilkan data dalam bentuk tabel. Elemen <table> berfungsi sebagai kontainer untuk elemen-elemen lain yang digunakan untuk mendefinisikan struktur dan isi tabel, seperti baris, kolom, dan sel. Elemen-elemen ini termasuk <tr> (table row) untuk baris, <th> (table header) untuk header kolom, dan <td> (table data) untuk sel data.

Contoh Tabel Sederhana

Nama Usia Kota
Andi 25 Jakarta
Budi 30 Surabaya
Citra 28 Bandung

Penjelasan Kode

  • <table border="1">: mendefinisikan tabel dengan garis pembatas (border) sebesar 1 piksel.
  • <tr>: digunakan untuk mendefinisikan baris dalam tabel.
  • <th>: digunakan untuk mendefinisikan sel header yang biasanya ditampilkan dengan teks tebal dan terpusat.
  • <td>: digunakan untuk mendefinisikan sel data dalam tabel.

Dengan kode di atas, hasil yang ditampilkan di browser adalah tabel sederhana dengan tiga kolom: "Nama," "Usia," dan "Kota," serta tiga baris data untuk masing-masing orang.
