Apa itu HTML?

HTML (Hypertext Markup Language) adalah bahasa markah yang digunakan untuk membuat dan menyusun konten di web. Ini adalah standar utama yang digunakan untuk membuat halaman web dan aplikasi web. HTML memungkinkan Anda untuk menentukan berbagai elemen di dalam halaman web, seperti teks, gambar, tautan, tabel, dan formulir.

Struktur HTML

Dokumen HTML memiliki struktur dasar yang terdiri dari beberapa elemen utama:

  • <!DOCTYPE html>: Menyatakan jenis dokumen dan versi HTML yang digunakan.
  • <html>: Elemen root yang membungkus seluruh konten halaman.
  • <head>: Bagian ini berisi metadata, seperti judul halaman (<title>), dan informasi lain seperti stylesheet atau skrip.
  • <body>: Bagian ini berisi semua konten yang akan ditampilkan di halaman web, seperti teks, gambar, dan tautan.

Membuat Heading

Heading digunakan untuk membuat judul atau subjudul di halaman web. HTML memiliki enam tingkat heading, dari <h1> hingga <h6>, dimana <h1> adalah yang paling penting dan <h6> yang paling tidak penting.

Contoh penggunaan heading:

        <h1>Ini adalah Heading 1</h1>
        <h2>Ini adalah Heading 2</h2>
        <h3>Ini adalah Heading 3</h3>
        <h4>Ini adalah Heading 4</h4>
        <h5>Ini adalah Heading 5</h5>
        <h6>Ini adalah Heading 6</h6>

Membuat Paragraf

Paragraf digunakan untuk menyusun teks menjadi bagian-bagian yang lebih kecil dan lebih mudah dibaca. Elemen <p> digunakan untuk membuat paragraf.

Contoh penggunaan paragraf:

        <p>Ini adalah paragraf pertama.</p>
        <p>Ini adalah paragraf kedua.</p>

Membuat List

List digunakan untuk menampilkan data atau informasi dalam bentuk daftar. Ada beberapa jenis list yang dapat digunakan dalam HTML:

Unordered List (Daftar Tak Berurutan)

Unordered list dibuat menggunakan elemen <ul> dan setiap item dalam daftar menggunakan elemen <li>. Daftar ini biasanya ditandai dengan bullet points (titik-titik).

Contoh penggunaan:

        <ul>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ul>

Hasil:

  • Item 1
  • Item 2
  • Item 3

Ordered List (Daftar Berurutan)

Ordered list dibuat menggunakan elemen <ol> dan setiap item dalam daftar menggunakan elemen <li>. Daftar ini biasanya ditandai dengan angka atau huruf.

Contoh penggunaan:

        <ol>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ol>

Hasil:

  1. Item 1
  2. Item 2
  3. Item 3

Description List (Daftar Deskripsi)

Description list digunakan untuk menampilkan daftar istilah dan deskripsinya. Dibuat menggunakan elemen <dl>, <dt> untuk istilah, dan <dd> untuk deskripsi.

Contoh penggunaan:

        <dl>
            <dt>HTML</dt>
            <dd>Hypertext Markup Language</dd>
            <dt>CSS</dt>
            <dd>Cascading Style Sheets</dd>
        </dl>

Hasil:

HTML
Hypertext Markup Language
CSS
Cascading Style Sheets

Preformatted Text (Teks Praformat)

Preformatted text digunakan untuk menampilkan teks dalam format yang sudah ditentukan sebelumnya, dengan mempertahankan spasi dan line breaks. Dibuat menggunakan elemen <pre>.

Contoh penggunaan:

        <pre>
            Ini adalah teks 
            yang diformat 
            dengan spasi dan line breaks.
        </pre>

Hasil:

        Ini adalah teks 
        yang diformat 
        dengan spasi dan line breaks.

Membuat List didalam List

Contoh Penggunaan:

    <ol>
        <li>List 1
            <ol>
                <li>Sub List 1</li>
                <li>Sub List 2</li>
            </ol>
        </li>
        <li>List 2</li>
        <li>List 3</li>
    </ol>

Hasil:

  1. List 1
    1. Sub List 1
    2. Sub List 2
  2. List 2
  3. List 3

Perbedaan utama antara elemen-elemen tersebut adalah:

  • <ul>: Digunakan untuk membuat daftar tak berurutan dengan bullet points.
  • <ol>: Digunakan untuk membuat daftar berurutan dengan angka atau huruf.
  • <li>: Digunakan sebagai item dalam <ul> atau <ol>.
  • <dl>: Digunakan untuk membuat daftar deskripsi dengan istilah dan deskripsinya.
  • <dt>: Digunakan untuk istilah dalam <dl>.
  • <dd>: Digunakan untuk deskripsi dalam <dl>.
  • <pre>: Digunakan untuk menampilkan teks dalam format yang sudah ditentukan sebelumnya, dengan mempertahankan spasi dan line breaks.

Membuat Element Link

Elemen link dalam HTML digunakan untuk membuat tautan ke halaman web lain, file, bagian lain dari halaman yang sama, atau lokasi lain di internet. Elemen link ini disebut <a>, yang merupakan singkatan dari "anchor"

Contoh Penggunaan:

        <p>Hubungi Saya
            <ul>
                <li><a href="https://wa.me/62822980055">WhatsApp</a></li>
                <li><a href="mailto:xxx@example.com">Email</a></li>
            </ul>
        <p>

Hasil:

Hubungi Saya

  • WhatsApp
  • Email
