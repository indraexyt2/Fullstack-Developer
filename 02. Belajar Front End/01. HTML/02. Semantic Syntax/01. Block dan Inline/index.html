Penjelasan Inline dan Block

Elemen Inline

Elemen inline adalah elemen yang tidak memulai baris baru dan hanya menempati ruang yang dibutuhkan oleh konten di dalamnya. Elemen-elemen ini umumnya digunakan untuk memformat teks atau konten kecil lainnya.

Contoh Elemen Inline:

  • <span>: Elemen ini digunakan untuk membungkus teks atau elemen lain tanpa mempengaruhi tata letak secara keseluruhan.
  • <a>: Tautan (link) yang biasanya digunakan untuk navigasi.
  • <img>: Elemen gambar yang disematkan di dalam teks.

Elemen Block

Elemen block adalah elemen yang memulai baris baru dan mengambil seluruh lebar kontainer induknya, sehingga biasanya digunakan untuk struktur dan tata letak halaman.

Contoh Elemen Block:

  • <div>: Elemen ini digunakan untuk membungkus konten dan membuat blok tata letak.
  • <p>: Elemen paragraf yang digunakan untuk teks.
  • <h1> - <h6>: Elemen heading untuk judul dengan berbagai ukuran.

Hubungan Antara Elemen Inline dan Block

Elemen inline dan block memiliki hubungan dalam hal bagaimana mereka digunakan untuk menyusun dan memformat konten di halaman web. Elemen block sering digunakan untuk struktur utama, sementara elemen inline digunakan untuk detail dan format dalam elemen block.
