Tabel Semantik HTML

Dalam HTML, tabel semantik digunakan untuk menyajikan data dalam format yang terstruktur, memudahkan pemahaman oleh manusia dan mesin (seperti mesin pencari dan pembaca layar). Tabel semantik menggunakan elemen-elemen berikut:

  • <table>: Elemen utama yang mendefinisikan tabel.
  • <thead>: Elemen yang digunakan untuk mendefinisikan bagian header dari tabel.
  • <tbody>: Elemen yang digunakan untuk mendefinisikan bagian tubuh (konten) dari tabel.
  • <tfoot>: Elemen yang digunakan untuk mendefinisikan bagian footer dari tabel.
  • <tr>: Elemen yang digunakan untuk mendefinisikan baris dalam tabel.
  • <th>: Elemen yang digunakan untuk mendefinisikan sel header dalam tabel.
  • <td>: Elemen yang digunakan untuk mendefinisikan sel data dalam tabel.

Contoh Kode HTML

Berikut adalah contoh kode HTML yang menunjukkan cara menggunakan elemen-elemen tabel semantik:

        
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Contoh Tabel Semantik</title>
            </head>
            <body>
                <table>
                    <thead>
                        <tr>
                            <th>Nama</th>
                            <th>Umur</th>
                            <th>Profesi</th>
                        </tr>
                    </thead>
                    <tbody>
                        <tr>
                            <td>Ali</td>
                            <td>25</td>
                            <td>Insinyur</td>
                        </tr>
                        <tr>
                            <td>Budi</td>
                            <td>30</td>
                            <td>Dokter</td>
                        </tr>
                        <tr>
                            <td>Citra</td>
                            <td>28</td>
                            <td>Guru</td>
                        </tr>
                    </tbody>
                    <tfoot>
                        <tr>
                            <td colspan="3">Data per Januari 2024</td>
                        </tr>
                    </tfoot>
                </table>
            </body>
            </html>

Hasilnya:

Contoh Tabel Semantik
Nama Umur Profesi
Ali 25 Insinyur
Budi 30 Dokter
Citra 28 Guru
Data per Januari 2024
